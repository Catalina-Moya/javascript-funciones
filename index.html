Funciones

Si presionas el cuadrado podrás cambiar el color de fondo.

hola

Si precionas cada cuadrado, se pintaran de color negro.

Si presionas las siguientes teclas (a-s-d), cambiarás el color del cuadrado.

La letra A guardará el color rosado.

La letra S guardará el color naranjo.

La letra D guardará el color celeste.

Al presionar las teclas (q-w-e), agregarás un cuadrado con los siguentes colores

La letra Q guardará el color morado.

La letra W guardará el color gris.

La letra E guardará el color café.
